# 🎉 CoinPilot v1.0.0 - Initial Release

**CoinPilot** is an offline cryptocurrency portfolio management CLI tool for privacy-conscious traders.

## ✨ Key Features

🔒 **100% Offline** - No internet required, your data stays local  
📊 **Trade Tracking** - Record BUY/SELL trades across multiple exchanges  
💰 **Auto Calculations** - Automatic position and P&L calculations  
📁 **CSV Storage** - Simple, readable data format you control  
🖥️ **Cross-Platform** - Windows, macOS, and Linux support  

## 🚀 Quick Start

```bash
# Add a trade
./coinpilot trade add --type BUY --symbol BTC --quantity 0.1 --price 45000

# View positions  
./coinpilot position list

# List trades
./coinpilot trade list
```

## 📦 Downloads

Choose the right version for your system:

### Windows
- **coinpilot-windows-x64.exe** - For 64-bit Windows (recommended)
- **coinpilot-windows-x86.exe** - For 32-bit Windows

### macOS  
- **coinpilot-macos-universal** - For all Macs (Intel + Apple Silicon)
- **coinpilot-macos-intel** - Intel Macs only
- **coinpilot-macos-arm64** - Apple Silicon (M1/M2) only

### Linux
- **coinpilot-linux-x64** - For 64-bit Linux
- **coinpilot-linux-arm64** - For ARM64 Linux

## 📋 Installation

1. Download the file for your platform
2. Make executable (Linux/macOS): `chmod +x coinpilot-*`  
3. Run: `./coinpilot-* --help`
